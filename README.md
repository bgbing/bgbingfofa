# bgbingfofa
fofa采集工具

如何使用？

python3 bgbingfofa.py -e/--email email -k/--key key -s/--size size

-e/--email 后面对应的是fofa邮箱  

-k/--key 后面对应的是fofakey

-s/--size 后面对应查询数量100/1000/10000
